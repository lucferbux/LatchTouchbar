# Latch Touchbar
Implementation of Latch by Elevenpaths in the Touchbar of a Macbook Pro.

## Requirements
 * Mac with touchbar and macOS 10.12.2 or higher
 * XCode 8 or higher
 * Latch account with and Application Id
 
## Installation
 * Download zip
 * Run Cocoapods
 * Build and execute in mac
